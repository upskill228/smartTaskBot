🚀 Projeto: TaskBot Inteligente

Criar um assistente que gere o workflow do utilizador via Linguagem Natural

1. Objetivo do Projeto

Construir uma aplicação (React ou Vanilla JS) que integre o Gemini 3 Flash para gerir tarefas de forma dinâmica. O Bot deve interpretar mensagens, manter o contexto da conversa e devolver JSON estruturado para atualizar a UI.

2. Funcionalidades Core (Obrigatórias)

💬 Chat & Streaming

  • Respostas em tempo real (Stream)
  • Histórico de mensagens (Contexto)
  • Interface estilo "Modern Chat"

📊 Gestão de Dados

  • Extração de Tarefa, Prioridade e Data
  • Lista de tarefas interativa (CRUD visual)
  • Filtros automáticos por intenção

3. Casos de Uso: Ecossistema ClickUp

O Bot deve ser capaz de mapear a linguagem informal para a estrutura organizacional que estudámos:

CRIAÇÃO AVANÇADA

"Adiciona uma tarefa urgente para rever o código no Espaço de Desenvolvimento para amanhã."

{ "action": "CREATE", "task": "Rever código", "priority": "URGENT", "space": "Desenvolvimento", "dueDate": "2026-05-05" }
EDIÇÃO POR CONTEXTO

"Afinal, muda a prioridade dessa tarefa para média e atribui-a ao Miguel."

{ "action": "UPDATE", "target": "last_task", "priority": "MEDIUM", "assignee": "Miguel" }

4. Sugestões de Projetos Alternativos

Podem adaptar a tecnologia de extração JSON e Streaming para outros domínios:

🍎 NutriBot Specialist

O utilizador descreve a refeição e o bot extrai macros e calorias, atualizando um diário alimentar.

{ "alimento": "Ovos", "kcal": 140, "proteina": "12g" }

💰 Finance Guard

Registo de despesas por voz ou texto. O bot categoriza gastos e atualiza o saldo do utilizador.

{ "valor": 45.50, "categoria": "Lazer", "local": "Cinema" }

✈️ Travel Planner

O utilizador diz para onde quer ir e o que quer ver; o bot cria um itinerário estruturado em lista.

{ "dia": 1, "local": "Torre Eiffel", "transporte": "Metro" }

5. Funcionalidades Opcionais (Bónus)

🎙️ Speech-to-Text

Submeter prompts através de voz usando a Web Speech API.

🛠️ Tool Use / Function Calling

Permitir que a IA execute funções reais (ex: apagar do LocalStorage).

🎨 Temas Dinâmicos

O Bot muda as cores da interface dependendo do humor da conversa.
